Command structure for Spawn Items:

ark -
admincheat giveslotitem "Blueprint'Item ID'" 1 500

Item ID get here: https://evolutionsofgame.github.io/ark-gogs/itemID.html

500 is Quantity of item spawn

ark - : any character but must above admincheat

For example:

arkGod-

admincheat giveslotitem "Blueprint'/Game/PrimalEarth/CoreBlueprints/Items/Structures/Misc/PrimalItemStructure_TekTeleporter.PrimalItemStructure_TekTeleporter_C'" 1 2

Command structure for Spawn Creatures:

 

*Method 1: Random level creatures + tamed

 

ARK
Admincheat SpawnActorTamed "Blueprint'Creature ID'" 10 1000 1000

 

ARK : any character but must above admincheat

 

Creature ID get from: https://evolutionsofgame.github.io/ark-gogs/CreaturesID.html

For example:

 

ARK
Admincheat SpawnActorTamed "Blueprint'/Game/PrimalEarth/Dinos/Gorilla/Gorilla_Character_BP.Gorilla_Character_BP_C'" 10 1000 1000

 

*Method 2: Custom level creatures + tamed

 

ARK
Admincheat GMSummon CreatureTag levelwild

 

ARK : any character but must above admincheat

 

CreatureTag: get from Creature ID example ID /Game/PrimalEarth/Dinos/Argentavis/Boss_Argent_Character_BP.Boss_Argent_Character_BP_C, creature tag : Boss_Argent_Character_BP_C

 

levelwild: level of creature when wild. Level after tame or after spawn of this command, it will bonus 50% level when wild, so if you want get creature level 540 = 360 + 50% * 360. you need put levelwild = 360

 

For Example:

 

ARK
Admincheat GMSummon Argent_Character_BP_C 360

 

you can get creature tamed level 540 after this commands

 

*Method 3: Random level creatures + Wild

similar GMSummon command but dont have level creature

 

ARK
Admincheat Summon CreatureTag

 

ARK : any character but must above admincheat

 

CreatureTag: get from Creature ID example ID /Game/PrimalEarth/Dinos/Argentavis/Boss_Argent_Character_BP.Boss_Argent_Character_BP_C, creature tag : Boss_Argent_Character_BP_C

 

For Example:

 

ARK
Admincheat Summon Argent_Character_BP_C
